# Momentum Mod

Shows your current velocity

## Integration

### How to toggle the mod

```java
public void enableMomentumMod(Player viewer, boolean value) {
    ModMomentum.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SCALE`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `0.5F`
        - Maximum: `1.5F`

- __`USE_GROUND_SPEED`__
    - If enabled then Y velocity is not used in the final speed.
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`USE_AVERAGE_VELOCITY`__
    - If this is disabled then instant velocity is used
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`AVERAGING_PERIOD`__
    - Determines how many times a second velocity is calculated. Lower values will show a smoother velocity
    - Values
        - Type: `Integer`
        - Default: `8`
        - Minimum: `1`
        - Maximum: `50`

- __`TEXT_SHADOW`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BACKGROUND`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`STATIC_BACKGROUND_WIDTH`__
    - If this is disabled the background will change size with the text.
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BACKGROUND_WIDTH`__
    - Values
        - Type: `Integer`
        - Default: `56`
        - Minimum: `50`
        - Maximum: `62`

- __`BACKGROUND_HEIGHT`__
    - Values
        - Type: `Integer`
        - Default: `18`
        - Minimum: `10`
        - Maximum: `22`

- __`BRACKETS`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BORDER`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`BORDER_THICKNESS`__
    - Values
        - Type: `Float`
        - Default: `0.5F`
        - Minimum: `0.5F`
        - Maximum: `3.0F`

- __`TEXT_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFFFFFFFF`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`BACKGROUND_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0x6F000000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`BORDER_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0x9F000000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`
