# Nick Hider

Allows you to hide skins and usernames for yourself or others.

## Integration

### How to toggle the mod

```java
public void enableNickHider(Player viewer, boolean value) {
    ModNickHider.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`HIDE_NAME`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`HIDE_REAL_NAME`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`HIDE_OTHERS_NAMES`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`HIDE_OWN_SKIN`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`USE_REAL_SKIN`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`HIDE_OTHERS_SKIN`__
    - Values
        - Type: `Boolean`
        - Default: `false`
