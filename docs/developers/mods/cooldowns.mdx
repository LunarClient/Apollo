# Cooldowns

Allows servers to display items or abilities that are on cooldown on the HUD.

## Integration

### How to toggle the mod

```java
public void enableCooldowns(Player viewer, boolean value) {
    ModCooldowns.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SCALE`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `0.5F`
        - Maximum: `1.5F`
