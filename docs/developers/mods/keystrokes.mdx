# Key Strokes

Displays when your movement, mouse or space bar is pressed.

## Integration

### How to toggle the mod

```java
public void enableKeyStrokes(Player viewer, boolean value) {
    ModKeystrokes.ENABLED.set(viewer, value);
}
```

## Available options
