# Shiny Pots

Gives potions the full box shiny enchantment effect.

## Integration

### How to toggle the mod

```java
public void enableShinyPots(Player viewer, boolean value) {
    ModShinyPots.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`COLORED_POTIONS`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`RENDER_GLINT_BEHIND_POTION`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`RENDER_ENTIRE_SLOT`__
    - Values
        - Type: `Boolean`
        - Default: `false`
