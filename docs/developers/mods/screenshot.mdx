# Screenshot Uploader

Updates the screenshot behaviour allowing you to instantly open, upload or tweet the image.

## Integration

### How to toggle the mod

```java
public void enableScreenshotUploader(Player viewer, boolean value) {
    ModScreenshot.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`COPY_AUTOMATICALLY`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`UPLOAD_OPTION`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`TWEET_OPTION`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`COPY_OPTION`__
    - Values
        - Type: `Boolean`
        - Default: `true`
