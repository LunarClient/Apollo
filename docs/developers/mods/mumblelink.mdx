# Mumble Link

Uses mumble to create positional audio in the Minecraft world.

## Integration

### How to toggle the mod

```java
public void enableMumbleLink(Player viewer, boolean value) {
    ModMumbleLink.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`
