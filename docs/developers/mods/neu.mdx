# Not Enough Updates

Skyblock Mod made by Moulberry

## Integration

### How to toggle the mod

```java
public void enableNotEnoughUpdates(Player viewer, boolean value) {
    ModNeu.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`
