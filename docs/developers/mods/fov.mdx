# FOV Changer

Allows you to customize your field of view for different potion or speed states.

## Integration

### How to toggle the mod

```java
public void enableFOVChanger(Player viewer, boolean value) {
    ModFov.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`STATIC_F_O_V`__
    - Determines if your FOV changes as you move.
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`DEFAULT_FOV`__

        - Type: `Integer`
        - Default: `70`

        - Maximum: `110`
