# Minimap

Display a miniature map of the world on your HUD.

## Integration

### How to toggle the mod

```java
public void enableMinimap(Player viewer, boolean value) {
    ModMinimap.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SCALE`__
    - Values
        - Type: `Float`
        - Default: `1.0F`

        - Maximum: `1.5F`

- __`MAP_WIDTH`__
