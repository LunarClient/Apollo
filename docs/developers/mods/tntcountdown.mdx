# TNT Countdown

Displays the fuse countdown above TNT.

## Integration

### How to toggle the mod

```java
public void enableTNTCountdown(Player viewer, boolean value) {
    ModTntCountdown.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`ADJUST_FUSE_TIME`__
    - Offset the default tnt delay in ticks. 1 second is 20 ticks
    - Values
        - Type: `Integer`
        - Default: `0`
        - Minimum: `-80`
        - Maximum: `80`

- __`BACKGROUND`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`TEXT_SHADOW`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`STATIC_COUNTDOWN_COLOR`__
    - Should the color stay the same the entire time.
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFF00FF00`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`PREFIX_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFFFFFFFF`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`
