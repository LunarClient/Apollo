# Snaplook

When activated, allows you to change your camera perspective while a button is pressed.

## Integration

### How to toggle the mod

```java
public void enableSnaplook(Player viewer, boolean value) {
    ModSnaplook.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SMOOTH_CAMERA`__
    - When zoomed in the camera movement will move smoothly (cinematic camera)
    - Values
        - Type: `Boolean`
        - Default: `true`
