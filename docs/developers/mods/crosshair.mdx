# Crosshair

Replace the default Minecraft crosshair with a customizable crosshair.

## Integration

### How to toggle the mod

```java
public void toggleCrosshairExample(Player viewer, boolean value) {
    Optional<ApolloPlayer> apolloPlayerOpt = Apollo.getPlayerManager().getPlayer(viewer.getUniqueId());
    apolloPlayerOpt.ifPresent(apolloPlayer -> this.modSettingModule.getOptions().set(apolloPlayer, ModCrosshair.ENABLED, value));
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`CROSSHAIR_THICKNESS`__
    - Values
        - Type: `Integer`
        - Default: `1`
        - Minimum: `1`
        - Maximum: `5`

- __`CROSSHAIR_SIZE`__
    - Values
        - Type: `Integer`
        - Default: `4`
        - Minimum: `0`
        - Maximum: `8`

- __`CROSSHAIR_GAP`__
    - Values
        - Type: `Integer`
        - Default: `0`
        - Minimum: `0`
        - Maximum: `8`

- __`CROSSHAIR_DOT`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`CROSSHAIR_OUTLINE`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`OUTLINE_THICKNESS`__
    - Values
        - Type: `Float`
        - Default: `0.5F`
        - Minimum: `0.0F`
        - Maximum: `1.0F`

- __`OUTLINE_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0x88000000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`CUSTOM_SCALE`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFFFFFFFF`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`FRIENDLY_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFF33FF33`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`ENEMY_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFFFF3333`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`
