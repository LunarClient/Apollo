# PvP Info

Displays various PvP statistics

## Integration

### How to toggle the mod

```java
public void enablePvPInfo(Player viewer, boolean value) {
    ModPvpInfo.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`RESET_ON_WORLD_CHANGE`__
    - Values
        - Type: `Boolean`
        - Default: `false`
