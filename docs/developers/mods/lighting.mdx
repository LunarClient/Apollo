# Lighting

Allows you to configure or disable lighting effects, which can increase FPS.

## Integration

### How to toggle the mod

```java
public void enableLighting(Player viewer, boolean value) {
    ModLighting.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`FULL_BRIGHT`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BRIGHTNESS_BOOST`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `1.0F`
        - Maximum: `10.0F`
