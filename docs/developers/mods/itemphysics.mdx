# Item Physics

Replaces ground items with ones that have physics.

## Integration

### How to toggle the mod

```java
public void enableItemPhysics(Player viewer, boolean value) {
    ModItemPhysics.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`ROTATION_SPEED`__
    - Values
        - Type: `Float`
        - Default: `0.5F`
        - Minimum: `0.1F`
        - Maximum: `4.0F`
