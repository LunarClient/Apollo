# Item Counter

Displays how many of the items you have selected are currently in your inventory, on the HUD.

## Integration

### How to toggle the mod

```java
public void enableItemCounter(Player viewer, boolean value) {
    ModItemCounter.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SCALE`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `0.5F`
        - Maximum: `1.5F`
