# Scoreboard

Allows you to hide, move, or customize the Minecraft scoreboard.

## Integration

### How to toggle the mod

```java
public void toggleScoreboardExample(Player viewer, boolean value) {
    Optional<ApolloPlayer> apolloPlayerOpt = Apollo.getPlayerManager().getPlayer(viewer.getUniqueId());
    apolloPlayerOpt.ifPresent(apolloPlayer -> this.modSettingModule.getOptions().set(apolloPlayer, ModScoreboard.ENABLED, value));
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SCALE`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `0.5F`
        - Maximum: `1.5F`

- __`NUMBERS`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`TEXT_SHADOW`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`BORDER_THICKNESS`__
    - Values
        - Type: `Float`
        - Default: `0.5F`
        - Minimum: `0.5F`
        - Maximum: `3.0F`

- __`BORDER`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`BACKGROUND_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0x50000000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`BORDER_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0x9F000000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`
