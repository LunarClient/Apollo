# Direction HUD

Display your cardinal direction on the HUD.

## Integration

### How to toggle the mod

```java
public void toggleDirectionHUDExample(Player viewer, boolean value) {
    Optional<ApolloPlayer> apolloPlayerOpt = Apollo.getPlayerManager().getPlayer(viewer.getUniqueId());
    apolloPlayerOpt.ifPresent(apolloPlayer -> this.modSettingModule.getOptions().set(apolloPlayer, ModDirectionHud.ENABLED, value));
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SCALE`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `0.5F`
        - Maximum: `1.5F`

- __`WIDTH`__
    - Values
        - Type: `Float`
        - Default: `280.0F`
        - Minimum: `168.0F`
        - Maximum: `448.0F`

- __`TEXT_SHADOW`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`BOLD_DIRECTIONS`__
    - Show the cardinal directions in BOLD.
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`BACKGROUND`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`BORDER`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`BORDER_THICKNESS`__
    - Values
        - Type: `Float`
        - Default: `0.5F`
        - Minimum: `0.5F`
        - Maximum: `3.0F`

- __`USE_LEGACY_STYLE`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SHOW_MARKER`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SHOW_MARKER_VALUE`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BACKGROUND_COLOR`__
    - Values
        - Type: `Integer`

- __`BORDER_COLOR`__

- __`DIRECTION_COLOR`__
    - Values

        - Maximum: `0x7FFFFFFF`
