# Zoom

Allows you to smoothly zoom in and out.

## Integration

### How to toggle the mod

```java
public void toggleZoomExample(Player viewer, boolean value) {
    Optional<ApolloPlayer> apolloPlayerOpt = Apollo.getPlayerManager().getPlayer(viewer.getUniqueId());
    apolloPlayerOpt.ifPresent(apolloPlayer -> this.modSettingModule.getOptions().set(apolloPlayer, ModZoom.ENABLED, value));
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`TOGGLE_KEY_ZOOM`__
    - Quickly pressing and releasing the key will toggle zoom
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SMOOTH_CAMERA`__
    - When zoomed in the camera movement will move smoothly (cinematic camera)
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SMOOTH_ZOOM`__
    - Add a transition when zooming in and out.
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`VARIABLE_ZOOM`__
    - Adjust the zoom depth using Mouse Scroll Wheel
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`ZOOM_DIVISOR`__
    - Change the initial zoom depth.
    - Values
        - Type: `Integer`
        - Default: `4`
        - Minimum: `2`
        - Maximum: `10`
