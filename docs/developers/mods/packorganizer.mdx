# Pack Organizer

Allows you to organize your resource packs into folders

## Integration

### How to toggle the mod

```java
public void enablePackOrganizer(Player viewer, boolean value) {
    ModPackOrganizer.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SHOW_INCOMPATIBLE`__
    - Values
        - Type: `Boolean`
        - Default: `true`
