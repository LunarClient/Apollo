# NameTags

A mod that allows you to modify nametags.

## Integration

### How to toggle the mod

```java
public void enableNameTags(Player viewer, boolean value) {
    ModNametag.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`NAMETAG_SHADOW`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`NAMETAG`__
    - An option to see your own nametag in third person
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`NAMETAG_LOGO`__
    - An option to toggle LC logos in nametags
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`TOGGLE_NAMETAGS`__
    - Determines if we should send the message when nametags are toggled.
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`NAMETAG_BACKGROUND_OPACITY`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `0.0F`
        - Maximum: `1.0F`
