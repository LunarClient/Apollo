# Menu Blur

Blurs the background behind menus.

## Integration

### How to toggle the mod

```java
public void enableMenuBlur(Player viewer, boolean value) {
    ModMenuBlur.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BLUR_STRENGTH`__
    - Values
        - Type: `Integer`
        - Default: `4`
        - Minimum: `0`
        - Maximum: `20`

- __`BLUR_LUNAR_TOGGLE`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BLUR_LUNAR_BACKGROUND`__
    - Values
        - Type: `Integer`
        - Default: `0x0`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`BLUR_INVENTORY_TOGGLE`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BLUR_INVENTORY_BACKGROUND`__
    - Values
        - Type: `Integer`
        - Default: `0x6F000000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`BLUR_PAUSE_TOGGLE`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BLUR_PAUSE_BACKGROUND`__
    - Values
        - Type: `Integer`
        - Default: `0x6F000000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`
