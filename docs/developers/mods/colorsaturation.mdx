# Color Saturation

## Integration

### How to toggle the mod

```java
public void toggleColorSaturationExample(Player viewer, boolean value) {
    Optional<ApolloPlayer> apolloPlayerOpt = Apollo.getPlayerManager().getPlayer(viewer.getUniqueId());
    apolloPlayerOpt.ifPresent(apolloPlayer -> this.modSettingModule.getOptions().set(apolloPlayer, ModColorSaturation.ENABLED, value));
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`COLOR_SATURATION_HUE`__
    - Values
        - Type: `Float`
        - Default: `0.0F`
        - Minimum: `0.0F`
        - Maximum: `10.0F`

- __`COLOR_SATURATION_SATURATION`__
    - Values
        - Type: `Float`
        - Default: `5.0F`
        - Minimum: `0.0F`
        - Maximum: `10.0F`

- __`COLOR_SATURATION_BRIGHTNESS`__
    - Values
        - Type: `Float`
        - Default: `5.0F`
        - Minimum: `0.0F`
        - Maximum: `10.0F`

- __`COLOR_SATURATION_CONTRAST`__
    - Values
        - Type: `Float`
        - Default: `5.0F`
        - Minimum: `0.0F`
        - Maximum: `10.0F`
