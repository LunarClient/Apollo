# Saturation

Saturation HUD with AppleSkin features

## Integration

### How to toggle the mod

```java
public void enableSaturation(Player viewer, boolean value) {
    ModSaturation.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SHOW_SATURATION_OVERLAY`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SHOW_APPLESKIN_TOOLTIP`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SHOW_HELD_ITEM_HUNGER`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SHOW_HELD_ITEM_SATURATION`__
    - Values
        - Type: `Boolean`
        - Default: `true`
