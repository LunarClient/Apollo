# 2D Items

Render blocks and items in a 2D perspective or with sprites.

## Integration

### How to toggle the mod

```java
public void enable2DItems(Player viewer, boolean value) {
    Mod2dItems.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`
