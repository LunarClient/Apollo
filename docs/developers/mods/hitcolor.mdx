# Hit Color

Changes the color of the damage indicator.

## Integration

### How to toggle the mod

```java
public void enableHitColor(Player viewer, boolean value) {
    ModHitColor.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`HIT_ARMOR_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0x66990000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`SHOULD_COLOR_ARMOR`__
    - Values
        - Type: `Boolean`
        - Default: `true`
