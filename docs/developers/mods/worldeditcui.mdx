# WorldEdit CUI

A client-side user interface for the WorldEdit plugin allowing you to see your selected region in-game in real time.

## Integration

### How to toggle the mod

```java
public void enableWorldEditCUI(Player viewer, boolean value) {
    ModWorldeditCui.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`POSITION_ONE_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFF0000FF`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`POSITION_TWO_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFFFF0000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`OUTLINE_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFF00FF00`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`
