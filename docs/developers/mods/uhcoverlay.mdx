# UHC Overlay

Resize items that are commonly needed in UHC to make spotting them easier.

## Integration

### How to toggle the mod

```java
public void enableUHCOverlay(Player viewer, boolean value) {
    ModUhcOverlay.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`GOLD_INGOT_SCALE`__
    - Values
        - Type: `Float`
        - Default: `2.0F`
        - Minimum: `1.0F`
        - Maximum: `5.0F`

- __`GOLD_NUGGET_SCALE`__
