# Hypixel Bedwars

Various enhancements to assist you when playing Hypixel Bedwars

## Integration

### How to toggle the mod

```java
public void enableHypixelBedwars(Player viewer, boolean value) {
    ModHypixelBedwars.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`COLORED_BEDS`__
    - Recolor the beds in-game according to the team's color
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BW_HARDCORE_HEARTS`__
    - Replaces normal hearts with hardcore hearts once your bed is lost
    - Values
        - Type: `Boolean`
        - Default: `true`
