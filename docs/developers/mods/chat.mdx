# Chat

Customize your chat options to your liking.

## Integration

### How to toggle the mod

```java
public void enableChat(Player viewer, boolean value) {
    ModChat.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`UNLIMITED_CHAT`__
    - Choose whether or not you want to increase the message history in chat
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`STACK_MESSAGES`__
    - Choose whether or not you want to stack multiple of the same messages in chat
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`CHAT_HEIGHT`__
    - Moves the chat up 12 pixels so it doesn't block health bar
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`CHAT_BACKGROUND_OPACITY`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `0.0F`
        - Maximum: `1.0F`

- __`CHAT_SHADOW`__
    - Show text shadow on chat lines
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`DISABLE_CHAT`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`NO_CLOSE_MY_CHAT`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`STOP_PROFANE_MESSAGES`__

    - Values
        - Type: `Boolean`
        - Default: `false`

- __`INPUT_FIELD_OPACITY`__
    - Values
        - Type: `Float`
        - Default: `5.0F`
        - Minimum: `0.0F`
        - Maximum: `10.0F`

- __`CHAT_PING_SOUND`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SMOOTH_CHAT`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SMOOTH_CHAT_SPEED`__
    - Values
        - Type: `Integer`
        - Default: `3`
        - Minimum: `1`
        - Maximum: `10`

- __`CHAT_TIMESTAMPS`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`TIMESTAMP_ITALICS`__
    - Makes the timestamp italics
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`TIMESTAMP_BOLD`__
    - Makes the timestamp bold
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`TWELVE_HOUR_CLOCK`__
    - Use a 12-hour clock, or a 24-hour clock if disabled
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`SHOW_AM_PM`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SHOW_SECONDS`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SHOW_BRACKETS`__
    - Adds square brackets around the timestamp
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`COPY_CHAT`__
    - Copies the hovered chat message when holding the keybind and clicking.
    - Values
        - Type: `Boolean`
        - Default: `false`
