# Auto Text Hot Key

Allows you to set a key or key combination that will send a command or chat message when pressed.

## Integration

### How to toggle the mod

```java
public void enableAutoTextHotKey(Player viewer, boolean value) {
    ModAutoTextHotkey.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`
