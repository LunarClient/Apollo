# Hypixel Quickplay

Quickly join games on the Hypixel network

## Integration

### How to toggle the mod

```java
public void enableHypixelQuickplay(Player viewer, boolean value) {
    ModQuickplay.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`
