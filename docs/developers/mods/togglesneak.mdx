# Toggle Sneak/Sprint

Make the sprinting and sneaking keys toggleable instead of needing to be held down.

## Integration

### How to toggle the mod

```java
public void toggleToggleSneakSprintExample(Player viewer, boolean value) {
    Optional<ApolloPlayer> apolloPlayerOpt = Apollo.getPlayerManager().getPlayer(viewer.getUniqueId());
    apolloPlayerOpt.ifPresent(apolloPlayer -> this.modSettingModule.getOptions().set(apolloPlayer, ModToggleSneak.ENABLED, value));
}
```

## Available options

- __`ENABLED`__
    - Config Key: `enabled`
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`TOGGLE_SPRINT`__
    - Config Key: `toggle-sprint`
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`TOGGLE_SNEAK`__
    - Config Key: `toggle-sneak`
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`DOUBLE_TAP`__
    - Config Key: `double-tap`
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`TOGGLE_SNEAK_CONTAINER`__
    - Config Key: `toggle-sneak-container`
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`FLY_BOOST`__
    - Config Key: `fly-boost`
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`FLY_BOOST_AMOUNT`__
    - Config Key: `fly-boost-amount`
    - Values
        - Type: `Integer`
        - Default: `4`
        - Minimum: `2`
        - Maximum: `8`
