# Freelook

When activated, allows you to change your camera perspective without rotating your player.

## Integration

### How to toggle the mod

```java
public void enableFreelook(Player viewer, boolean value) {
    ModFreelook.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`INVERT_PITCH`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`INVERT_YAW`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`TOGGLE_KEY_FREELOOK`__
    - Quickly pressing and releasing the key will toggle freelook
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SMOOTH_CAMERA`__
    - When zoomed in the camera movement will move smoothly (cinematic camera)
    - Values
        - Type: `Boolean`
        - Default: `true`
