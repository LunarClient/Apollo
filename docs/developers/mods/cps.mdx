# CPS

Displays your clicking speed on the HUD.

## Integration

### How to toggle the mod

```java
public void toggleCPSExample(Player viewer, boolean value) {
    Optional<ApolloPlayer> apolloPlayerOpt = Apollo.getPlayerManager().getPlayer(viewer.getUniqueId());
    apolloPlayerOpt.ifPresent(apolloPlayer -> this.modSettingModule.getOptions().set(apolloPlayer, ModCps.ENABLED, value));
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SCALE`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `0.5F`
        - Maximum: `1.5F`

- __`RIGHT_CLICK`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`SHOW_CPSTEXT`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`LINE_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFF202020`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`TEXT_SHADOW`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BACKGROUND`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`STATIC_BACKGROUND_WIDTH`__
    - If this is disabled the background will change size with the text.
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BACKGROUND_WIDTH`__
    - Values
        - Type: `Integer`
        - Default: `56`
        - Minimum: `40`
        - Maximum: `62`

- __`BACKGROUND_HEIGHT`__
    - Values
        - Type: `Integer`
        - Default: `18`
        - Minimum: `10`
        - Maximum: `22`

- __`BRACKETS`__
    - Values
        - Type: `Boolean`
        - Default: `true`

- __`BORDER`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`BORDER_THICKNESS`__
    - Values
        - Type: `Float`
        - Default: `0.5F`
        - Minimum: `0.5F`
        - Maximum: `3.0F`

- __`TEXT_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xFFFFFFFF`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`BACKGROUND_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0x6F000000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`

- __`BORDER_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0x9F000000`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`
