# Fog Customizer

Allows you to adjust the strength of fog for various fog types

## Integration

### How to toggle the mod

```java
public void enableFogCustomizer(Player viewer, boolean value) {
    ModFog.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`WATER_FOG_DENSITY`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `0.0F`
        - Maximum: `1.8F`

- __`RENDER_DISTANCE_FOG_DENSITY`__
    - Values
        - Type: `Float`
        - Default: `1.0F`
        - Minimum: `0.0F`
        - Maximum: `1.95F`

- __`RENDER_DISTANCE_FOG_COLOR_TOGGLE`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`RENDER_DISTANCE_FOG_COLOR`__
    - Values
        - Type: `Integer`
        - Default: `0xC0D8FF`
        - Minimum: `0x80000000`
        - Maximum: `0x7FFFFFFF`
