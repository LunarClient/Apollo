# Motion Blur

Blur your surroundings when moving.

## Integration

### How to toggle the mod

```java
public void enableMotionBlur(Player viewer, boolean value) {
    ModMotionBlur.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`OLD_BLUR`__
    - Values
        - Type: `Boolean`
        - Default: `false`

- __`VALUE`__
    - Values
        - Type: `Integer`
        - Default: `1`
        - Minimum: `1`
        - Maximum: `10`
