# Better Sounds

Modify the volume of any sound in the game

## Integration

### How to toggle the mod

```java
public void enableBetterSounds(Player viewer, boolean value) {
    ModSoundChanger.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`
