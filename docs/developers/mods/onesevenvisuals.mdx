# 1.7 Visuals

Revert certain visuals and animations to how they behaved on 1.7.

## Integration

### How to toggle the mod

```java
public void enable1_7Visuals(Player viewer, boolean value) {
    ModOneSevenVisuals.ENABLED.set(viewer, value);
}
```

## Available options

- __`ENABLED`__
    - Values
        - Type: `Boolean`
        - Default: `false`
