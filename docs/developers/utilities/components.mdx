# Components

## Overview

Apollo implements a streamlined version of [Minecraft's JSON text format](https://minecraft.fandom.com/wiki/Raw_JSON_text_format) optimized for consistent rendering on all Minecraft versions. Most notably, features like translatable text are removed in favor of server-side localizations, and arbitrary RGB colors (added in vanilla Minecraft 1.16) are enabled on all versions of Lunar Client.

## `Component` Builder

```java
public class Component {

    /**
     * Returns the renderable string {@link String} content.
     *
     * @return the renderable string content
     * @since 1.0.0
     */
    String content;

    /**
     * Returns the renderable string {@link Color}.
     *
     * @return the renderable string color
     * @since 1.0.0
     */
    Color color;

    /**
     * Returns a {@link List} of {@link TextDecorators}.
     *
     * @return the renderable string decorators
     * @since 1.0.0
     */
    List<TextDecorators> decorators;

    /**
     * Returns a {@link List} of {@link Component}.
     *
     * @return the renderable string children
     * @since 1.0.0
     */
    List<Component> children;

    /**
     * Represents the text decorator.
     *
     * @since 1.0.0
     */
    public enum TextDecorators {
        OBFUSCATED,
        BOLD,
        STRIKETHROUGH,
        UNDERLINED,
        ITALIC
    }

}
```

### Sample Code

```java
public static Component componentExample() {
    return Component.builder()
        .content("Hello world!")
        .color(Color.GREEN)
        .decorators(Lists.newArrayList(Component.TextDecorators.BOLD, Component.TextDecorators.UNDERLINED))
        .children(Lists.newArrayList())
        .build();
}
```

## `Component` Options

`String content` needs to contain the string of text you want displayed.

```java
.content("Hello world!")
```

`Color color` is how you dictate the color of the text. See the [colors page](/apollo/developers/utilities/colors) for more. Arbitrary RGB colors are supported on all Minecraft versions with Lunar Client.

```java
.color(Color.GREEN)
```

`List<TextDecorators> decorators` are the text decorators you want to apply to the `content` string. Read the section below to find out more about `TextDecorators`

```java
.decorators(List.of(Component.TextDecorators.BOLD, Component.TextDecorators.UNDERLINED))
```

`List<Component> children` the list of extra json components to be displayed after the previous `content`. The style is inherited from the parameters above; `Color` & `TextDecorators`

```java
.children(Lists.newArrayList())
```


## `TextDecorators` Predefined

* `OBFUSCATED`
* `BOLD`
* `STRIKETHROUGH`
* `UNDERLINED`
* `ITALIC`
